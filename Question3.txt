Letra A = 1, 3, 5, 7, 9; sequência de números ímpares;

Letra B = 2, 4, 8, 16, 32, 64, 128; cada número é o dobro do número anterior;

Letra C = 0, 1, 4, 9, 16, 25, 36, 49; parece ser a sequência dos quadrados perfeitos;

Letra D = 4, 16, 36, 64, 100; parece ser a sequência dos quadrados perfeitos, pegando somente os números pares;

Letra E = 1, 1, 2, 3, 5, 8, 13; sequência de Fibonacci;

Letra F = 2,10, 12, 16, 17, 18, 19, 19,5; (2 pra 10 dá 8 de diferença, 12 pra 16 dá 4 de diferença...);
